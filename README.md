# Girls Band Cry (GBC) Anki Deck

这是一个用于通过动漫 **《Girls Band Cry》 (GBC)** 学习日语的 Anki 卡组项目。本项目旨在通过沉浸式的听力与台词练习，帮助学习者掌握地道的日语表达。

> 当前版本重点覆盖 **EP02** (第2集) 的核心内容。

## 📁 内容介绍 (Content)

- **Deck Name**: GBC v0.1
- **核心内容**:
  - 包含 **GBC EP02** 的所有关键台词。
  - 收录 **OP** (片头曲)、**IN** (插曲)、**ED** (片尾曲) 的听力特训卡。
  - 台词中的重点词汇与语法解析。
- **卡片类型**:
  - 🎧 **Listening 模板**: 专注于听写与影子跟读。
  - 📖 **Study 模板**: 专注于词汇记忆与语法解析。

## 🚀 如何使用 (Installation)

1. 在本页面的右侧 **"Releases"** 区域，点击最新的版本 (例如 v0.1)。
2. 下载列表中的 `.apkg` 文件 (如 `GBC v0.1.apkg`)。
3. 双击文件，或打开 Anki 选择菜单栏的“文件 -> 导入”。

## 🐛 反馈与勘误 (Feedback)

这是一个开源项目，难免会有疏漏。如果你在使用过程中发现：

- 日语台词或翻译有误
- 单词释义不准确
- 样式显示异常

请直接点击页面上方的 **[Issues]** 标签，建立一个新的 Issue 进行反馈。非常欢迎你的指正！

## 🛠️ 开发者指南 (For Developers)

如果你想自定义或修改卡片样式，源码位于 `src/` 目录下：

- `study.css`: 学习 (Study) 模板的样式表
- `listening.css`: 听力 (Listening) 模板的样式表
- `*.html`: 各模板正反面的 HTML 结构代码
